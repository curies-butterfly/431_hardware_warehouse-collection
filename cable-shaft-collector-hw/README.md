# cable-shaft-collector-hw

线缆竖井采集器硬件 - 线缆竖井环境数据采集硬件设计，包含原理图、PCB 及相关设计文件。

## 项目信息

- **项目名称**: cable-shaft-collector-hw
- **仓库分支**: `cable-shaft-collector-hw`
- **设计工具**: KiCad

## 目录结构

```
cable-shaft-collector-hw/
+-- cable-shaft-collector.kicad_pcb   (PCB 文件)
+-- cable-shaft-collector.kicad_sch   (主原理图)
+-- core.kicad_sch                    (核心子原理图)
+-- power.kicad_sch                   (电源子原理图)
+-- tpunb.kicad_sch                   (TPUNB 子原理图)
+-- transceiver.kicad_sch             (收发器子原理图)
+-- 4g.kicad_sch                      (4G 通信子原理图)
+-- local_lib.kicad_sym               (本地元件库)
+-- local_lib.pretty/                 (本地封装库)
+-- ... (其他设计文件)
```

## 说明

本分支包含线缆竖井采集器项目的所有设计文件，包括 KiCad 格式的原理图、PCB 布局及元件库。
